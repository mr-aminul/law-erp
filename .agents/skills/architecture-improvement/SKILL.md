# Architecture Improvement

Use this skill as the action-oriented entry point for `/architecture-review`.

## Workflow

1. Read `.agents/skills/architecture-review/SKILL.md`.
2. Map the current flow and existing conventions.
3. Recommend the smallest architecture improvement that reduces real friction.
4. Avoid speculative layers, new frameworks, or broad rewrites.
5. Include the checks that prove the improvement is safe.

## Output

Provide the current shape, the concrete improvement, the smallest next patch, and the verification command.
